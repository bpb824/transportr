Export GTFS stops to ESRI shapefile

Usage

exportStopShape(feedPath, outPath, shapeName, returnShape = FALSE)

Arguments

feedPath
The relative or absolute path to the folder containing the GTFS feed text files
outPath
The relative or absolute path to the folder where you would like to save the reuslting shapefile
shapeName
The filename for your resulting shapefile
returnShape
Boolean indicating if route shape should be returned

Value

Returns stops shape as SpatialPointsDataFrame if desired

Description

Export GTFS stops to ESRI shapefile
