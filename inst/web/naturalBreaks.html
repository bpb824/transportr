Calculate natural breaks for specified fields in table

Usage

naturalBreaks(table, fields, numBreaks)

Arguments

table
The data frame or matrix object with the data and fields you need to calculate breaks for
fields
A vector of character strings corresponding to the fields you need to calculate breaks for
numBreaks
The number of break points to calculate

Value

Data frame with break points for each field specified

Description

This function will calculate natural (Jenks) breaks for each specified field in the input table and return a table of the break points based on the number of breaks desired.
