transportr

transportr is an R package for querying, managing, exploring, and analyzing transportation/transportation-related data. For more information about the project, visit http://bigtransportdata.com/transportr. The transportr package is being documented using staticdocs at http://bigtransportdata.com/_transportr/docs/

Installation

This package is still in development, but you can get the most recent development version from Bryan's GitHub:

# install.packages("devtools")
devtools::install_github("bpb824/transportr")

Some functions in this package require API Keys.

Help topics

Google API Utilities

These functions are used to interact with Google APIs.

  • decodeLineR
    Google Polyline Decoder
  • getGoogleDirections
    Fetch Google directions from Directions API
  • lines2points
    Convert SpatialLines to coordinate data frame for map matching.
  • route2shape
    Convert decoded Google Directions polyline into SpatialLines object

GTFS Utilities

These functions are used to download and prepare General Transit Feed Specification (GTFS) data.

  • fetchFeed
    Fetch GTFS feed files
  • spatialTransitShapes
    Create SpatialLines from GTFS shapes.txt file
  • exportRouteShape
    Export GTFS routes to ESRI shapefile

General Use Functions

These functions are used to peform various intermediate data handling.

  • list2frame
    Converts Two-Level List to Data Frame
  • naturalBreaks
    Calculate natural breaks for specified fields in table
