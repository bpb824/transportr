Fetch Google directions from Directions API

Usage

getGoogleDirections(from, to, key, mode, departure = NULL, arrival = NULL, alternatives = FALSE, waypoints = NULL)

Arguments

from
Start location to fetch directions for (vector in [Lat Lng] format)
to
End location to fetch directions for (vector in [Lat Lng] format)
key
API key for Google Directions API (from Google Developers Console)
mode
Travel mode: one of four strings: 'driving', 'walking', 'bicycling', 'transit'
departure
Optional departure time of POSIXct class. Cannot specify both arrival and departure times.
arrival
Optional arrival time of POSIXct class. Cannot specify both arrival and departure times.
waypoints
Optional additional locations to travel through along route. A data frame or with 'lat' and 'lng' columns, and an observation for each destination.

Value

Encoded google directions string

Description

Fetch Google directions from Directions API
