Geocode a place name using Google Places API

Usage

geocode_place(placeString, key, output = "loc")

Arguments

placeString
A string describing the place you'd like to geocode
output
The output can either be a simple location ('loc') or a list with all response results ('all')

Value

Location or list of API results
